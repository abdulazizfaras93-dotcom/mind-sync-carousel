# MindSyncKW Carousel Generator

A simple web-based tool to generate professional carousels for WhatsApp business.

## Features

- Multiple carousel templates
- Responsive design
- Easy to use interface
- Export-ready output

## Templates

1. **Problem-Solution** - Show a problem and its solution
2. **Before/After** - Compare before and after scenarios
3. **Behind the Scenes** - Show behind the scenes content
4. **FAQ** - Answer frequently asked questions
5. **Tips & Tricks** - Share helpful tips and tricks

## How to Use

1. Select a template from the dropdown
2. Fill in the content for each slide
3. Click "Generate Carousel" to create your carousel
4. Copy and paste the generated HTML into your WhatsApp business profile

## Installation

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Start creating carousels!

## Customization

You can customize:
- Colors (modify CSS)
- Templates (edit JavaScript)
- Images (replace placeholder images)
- Content structure (modify HTML)

## Browser Support

- Chrome 50+
- Firefox 50+
- Safari 10+
